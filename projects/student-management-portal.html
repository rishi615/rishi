Simple Student Management Portal

My Role

As it is a personal portfolio project, I was the sole designer and developer. I designed and developed the entire portal myself.

Definition of Problem

The requirement is to develop a Web application which will read data from a .csv file and perform various operations like adding, removing, and displaying student's details.

Project Goals

To design and develop a simple student management portal for adding, removing, and displaying student's details from a csv file.

  3. Create a page to display students, which will display all the students data in an HTML page.
  4. Create a good design for visual satisfaction keeping in mind the usability of the site.

Solution

The simple Student Management Portal uses simple technologies like HTML, CSS, Python and Flask.

Procedure

  1. Create the CSV file (students.csv) to store the details of the Students.
  2. Write a Flask Server using Python, which redirects the user to different routes.
  3. The Flask Server should also be able to read, write and manipulate the data in the CSV file.
  4. Create a main HTML page (index.html) which redirects the user to the desired pages.
  5. Create an add-student.html page which has a form that collects the student data entered and sends it to the back-end sever to store in the students.csv file.
  6. Create a search-student.html page through which the user searches for students’ details using the student Id. The id is sent to the back-end server which then matches the id with the data in the students.csv file and a linear search algorithm is performed on the data to find the correct student record.
  7. Create a display-students.html page that displays the details of all the students recorded in the students.csv file.
