A Simple Portfolio Website for showing Projects and Resume

Web Design and Development
7 min read

Simple and user-friendly Website

Check it out!

My Role

As it is a personal portfolio project, I was the sole designer and developer. I designed and developed the entire website myself.

Background

I have always wanted to make a portfolio website for myself, so I finally took out some time to do it. I wanted to make a simple and user friendly website so I developed it from scratch using HTML, CSS and little of JavaScript.

Definition of Problem

Many people struggle to showcase themselves and their skills in the right way. People think that they need to be excellent at designing and coding for making portfolio sites.

Project Goals

To design and develop a simple portfolio website for beginners or non-tech users.

  • Design a simple site to help users showcase their skills and abilities.
  • The wesbite should be responsive, easy to navigate around and accessible across multiple devices.
  • Create a good design for visual satisfaction keeping in mind the usability of the site.

Solution

The Simple Portfolio site uses simple technologies like HTML, CSS and little bit of Javascript. No Frameworks, templates or even Bootstrap is used in order to reduce complexitiy for beginners.

Procedure

  1. Designing the site: A simple design consisting of the following sections in the respective order.
    • Navigation menu at the top
    • Heading the page
    • Main body of the page, article
    • Footer
    • Separate pages: About, Certifications, Projects.
  2. Turning the design into actual code:
    • Writing the HTML for the structure of the website.
    • Writing HTML for different pages of the website.
    • Writing the CSS for styling the web pages.
    • Including some JavaScript to add interactive elements and other functionalities.
